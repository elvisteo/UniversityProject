1) Open the project (UniversityProject) in Visual Studio 2022.
2) Run the project by clicking "Debug" > "Start Debugging"  in the top menu or by pressing F5.
3) The following APIs will be in Swagger:
   -(POST) /api/University: To create record
   -(GET) /api/University/GetAll: To retrieve all records
   -(GET) /api/University/{id}: To retrieve record by Id
   -(PUT) /api/University/{id}: To update record by Id
   -(DELETE) /api/University/{id}: To delete record by Id
   -(POST) /api/University/bookmark/{id}: To bookmark by Id
